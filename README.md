## 📓 Notebook Breakdown

### 01 — Pandas Series
Introduction to the foundational 1D data structure in Pandas. Covers creation, indexing, slicing, and basic operations on Series.

### 02 — Pandas DataFrames
Deep dive into 2D DataFrames — creation, column selection, row filtering, adding/dropping columns, and basic exploration.

### 03 — Missing Data
Techniques to detect, handle, and clean missing values and duplicate rows — the most common real-world data problem.

### 04 — Merging, Joining & Concatenation
Combining multiple DataFrames using `merge()`, `join()`, and `concat()` — essential for working with relational data.

### 05 — GroupBy & Aggregation
Splitting data into groups and applying aggregation functions like `sum`, `mean`, `count`, `min`, `max` to extract insights.

### 06 — Pivot Tables
Creating pivot tables to summarize and cross-tabulate data — similar to Excel pivot tables but faster and more powerful.

### 07 — Pandas Operations
Useful everyday Pandas operations — `apply()`, `map()`, `lambda functions`, string methods, and more.

### 08 — Project: Anime Feature Extraction 🎌
A mini project using `anime.csv` to practice feature extraction, filtering, and data transformation on a fun real-world dataset.

### 09 — Capstone Project: Countries 🌍
A full end-to-end data analysis project using `Countries.csv` — applying all Pandas skills learned throughout the course.
